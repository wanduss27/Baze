Cheat Launch Instructions
==========================

Step 1: Extract the Archive
---------------------------
1. **Download and extract the archive** containing the cheat to your computer.
2. Inside the archive, you will find the following files:
   - `NcCrack.exe`
   - `cheat.dll`
   - `config.ini`

Step 2: Preparation
-------------------
1. **Move the files** `cheat.dll` and `config.ini` to the game folder (where the main game executable file is located).

Step 3: Check Settings
----------------------
1. Ensure that the files `cheat.dll` and `config.ini` are in the same folder as the game's executable file.

Step 4: Launch the Cheat
------------------------
1. **Run** `NcCrack.exe`.
2. If everything is done correctly, you will see a notification about the successful activation of the cheat.

Step 5: Play the Game
---------------------
1. Launch the game and enjoy the new features.

Important!
----------
Do not close `NcCrack.exe` while playing the game.
